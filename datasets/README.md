# DiaTracker Datasets

This folder contains all datasets, scripts, and documentation for the DiaTracker project.

## 📁 Folder Structure

```
datasets/
├── documentation/          # All documentation and summaries
├── processed_data/         # Cleaned and processed datasets ready for use
├── raw_data/              # Original unprocessed data
├── scripts/               # Python scripts for data processing and model training
├── models/                # Trained ML models (generated by scripts)
└── README.md             # This file
```

## 📊 Datasets Overview

### Processed Data (`processed_data/`)

**1. nhanes_2021_2023_processed.csv**
- **Source**: NHANES 2021-2023 survey data
- **Participants**: 2,660
- **Features**: 19 (after cleaning and processing)
- **Purpose**: Training data for RF #1 and RF #2 models
- **Processing**: See `scripts/process_nhanes_2021_2023.py`

**2. gi_database_enriched.csv**
- **Source**: Foster-Powell (2002) + USDA FoodData Central
- **Foods**: 200 with complete nutrition
- **Features**: GI, GL, protein, fat, fiber, calories, carbs
- **Purpose**: Food database for meal predictions
- **Processing**: Fuzzy matching + manual verification

**3. food_lookup_database.csv**
- **Source**: Enriched GI database
- **Purpose**: Production food lookup for the application
- **Format**: Optimized for fast queries

**4. nhanes_with_gl.csv**
- **Source**: NHANES + Foster-Powell merged
- **Purpose**: Experimental dataset (not used in final models)
- **Note**: Research only

### Raw Data (`raw_data/`)

**1. Foster-Powell glycemic index table 2002/**
- Original PDF and extracted data from Foster-Powell et al. (2002)
- 201 foods with GI and GL values

**2. nhanes_2021_2023/**
- Original NHANES XPT files from CDC
- Demographics, examination, laboratory, and questionnaire data

**3. nhanes_xpt_csvs/**
- Converted CSV files from XPT format
- Intermediate processing step

**4. foster_powell_2002_raw_data.csv**
- Raw extracted data from Foster-Powell PDF
- Before enrichment with USDA data

**5. gi_database_needs_review.csv**
- Foods that need manual verification
- Low confidence matches (<0.8 similarity)

**6. gi_database_research_only.csv**
- Experimental food data
- Not used in production

## 🔧 Scripts (`scripts/`)

### Data Processing Scripts

**1. process_nhanes_2021_2023.py**
- Processes raw NHANES data
- Cleans and filters participants
- Applies ADA 2024 risk classification
- Output: `nhanes_2021_2023_processed.csv`

**2. extract_foster_powell_raw.py**
- Extracts data from Foster-Powell PDF
- Creates raw CSV with GI/GL values
- Output: `foster_powell_2002_raw_data.csv`

**3. create_foster_powell_raw_csv.py**
- Alternative extraction method
- Manual data entry verification

**4. convert_xpt_to_csv.py**
- Converts NHANES XPT files to CSV
- Makes data easier to work with

**5. merge_gi_with_nhanes.py**
- Experimental: Merges GI data with NHANES
- Output: `nhanes_with_gl.csv` (research only)

### Model Training Scripts

**6. train_rf_models.py** ⭐ **MAIN TRAINING SCRIPT**
- Trains both RF #1 and RF #2 models
- Performs 5-fold cross-validation
- Saves models to `models/` folder
- Generates performance reports

**7. compare_models.py**
- Compares different ML algorithms
- Tests Linear Regression, Decision Tree, Random Forest
- Outputs performance metrics

## 🤖 Models (`models/`)

Generated by `train_rf_models.py`:

**1. rf_glucose_predictor.pkl**
- RF #1: Glucose Predictor
- R² = 0.91
- Features: 8 (fasting glucose, carbs, BMI, fiber, fat, protein, age, gender)

**2. rf_risk_classifier.pkl**
- RF #2: Risk Classifier
- Accuracy = 100%
- Features: 5 (fasting glucose, BMI, age, gender, family history)

**3. model_metadata.json**
- Model versions, training dates, performance metrics
- Feature lists and preprocessing parameters
